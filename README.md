The project uses fundamental Spring Boot features like:
> Authentication
> 
> Authorization
> 
> Custom Error Handlers
> 
> Databases
> 
> Transactional Operations
> 
> Custom Logging
> 
> Custom Validators
> 
> Spring Beans, Components and Configurations
> 
> Project Lombok
> 
> Exceptions
